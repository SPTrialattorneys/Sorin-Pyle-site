# Digital Business Card Design Mockups - Test Options

## Overview
This directory contains 4 test mockup designs for the Sorin & Pyle digital business cards, showcasing different approaches to using the firm's brand colors (Blue #4076B4 and Orange #FF8A28).

Each mockup demonstrates a **no-scroll, single viewport** design optimized for both mobile and desktop viewing.

## How to Test

### Step 1: Open Files in Browser
1. Navigate to the `test-mockups` directory
2. Double-click each `.html` file to open in your default browser
3. OR right-click → "Open with" → Choose your preferred browser

### Step 2: Desktop Testing
1. **Full Screen View**: Press F11 (or Cmd+Ctrl+F on Mac) to see full-screen experience
2. **Centered Card**: Resize browser window to see card centering behavior
3. **Hover Effects**: Hover over buttons and contact items to see interactions
4. **Window Resize**: Drag window edge to test responsive breakpoints

### Step 3: Mobile Testing

**Option A: Browser DevTools (Recommended)**
1. Open Developer Tools (F12 or Cmd+Option+I)
2. Toggle device toolbar (Ctrl+Shift+M or Cmd+Shift+M)
3. Select device: iPhone 12 Pro, iPhone SE, Pixel 5, etc.
4. Test portrait and landscape orientations

**Option B: Actual Mobile Device**
1. Transfer files to phone OR upload to web server
2. Open in mobile browser (Safari, Chrome)
3. Test touch interactions and viewport sizing

### Step 4: Side-by-Side Comparison
1. Open all 4 options in separate browser tabs
2. Quickly switch between tabs using Ctrl+Tab
3. Take notes on which design feels:
   - Most professional
   - Most memorable
   - Best brand representation
   - Best visual hierarchy
   - Best color balance

## Evaluation Criteria

### Professional Appearance
- Does it convey legal authority and trustworthiness?
- Is it appropriate for criminal defense clients?
- Does it balance professional with approachable?

### Brand Representation
- Do both blue and orange feel integrated?
- Is one color overwhelming the other?
- Does it represent the "We Give A [EXPLETIVE]" brand personality?

### Visual Hierarchy
- Does your eye go to the photo first?
- Is the name clearly the most prominent text?
- Does the orange CTA button draw attention?
- Is contact info easily readable?

### Mobile Experience
- Is all content visible without scrolling?
- Are touch targets large enough (44px minimum)?
- Is text readable at mobile sizes?
- Do buttons feel easy to tap?

### Memorability
- Which design would you remember after seeing it once?
- Which stands out from typical legal business cards?
- Which feels unique to Sorin & Pyle specifically?

## Questions or Issues?

### Image Not Loading?
- Ensure you're viewing from the `test-mockups` directory
- Jonathan's photo should be at `../images/jonathan-2.avif`
- If broken, check that the images folder exists one level up

### Layout Issues?
- Try a different browser (Chrome, Firefox, Safari, Edge)
- Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)
- Ensure browser window is at least 320px wide

### Want to Modify?
- All CSS is inline in each HTML file
- Edit the `<style>` section to experiment
- Save and refresh browser to see changes
